# essora-linux
Essora Eos is a minimalist GNU/Linux distribution based on Devuan Excalibur, designed to be lightweight, fast, and fully customizable. It provides only the essentials—without bloatware or unnecessary configurations—so the user keeps full control.

Editions
- KDE Plasma (official edition)
- Openbox
- JWM
- MATE
- SonicDE

Key features
- Fast and lightweight base (Devuan Excalibur, no systemd)
- Multiple desktop editions to fit different hardware and preferences
- Multi-language support (12+ languages)
- Essential tools included (file manager, terminal, image viewer, audio player)
- Friendly for remastering and customization

Name origin
Essora is an invented name inspired by “Essence” (core) and “Ora” (Latin: hour/now). Eos is the Greek goddess of dawn, representing new beginnings and clarity. “Essora Eos” means “the new dawn of essentials.”
