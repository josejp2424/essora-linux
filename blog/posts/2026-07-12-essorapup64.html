EssoraPup64 with XLibre

A modern and lightweight Puppy Linux system with its own integrated applications

64-bit XLibre Linux Kernel 7.1.3 JWM EssoraWM DEB Flatpak AppImage Live and Frugal Installation

EssoraPup64 is a 64-bit distribution based on Puppy Linux that combines the XLibre graphical server, the lightweight JWM and EssoraWM window managers, and a complete collection of applications developed specifically for the Essora ecosystem.

The project aims to preserve the simplicity, flexibility and low resource usage that characterize Puppy Linux while providing a more modern, integrated and ready-to-use desktop experience.

EssoraPup64 can be used as a portable Live system, installed through a traditional Puppy-style frugal installation, or customized and remastered into a new ISO image.

EssoraPup64 keeps the lightweight philosophy of Puppy Linux while adding its own application store, installer, network tools, system utilities, multimedia applications and desktop integration.

Main features

  • 64-bit architecture
  • XLibre graphical server
  • Linux Kernel 7.1.3
  • Lightweight JWM and EssoraWM desktop environments
  • Support for DEB, Flatpak and AppImage applications
  • Custom graphical tools for system administration
  • Live system support
  • Frugal installation support
  • BIOS and UEFI installation support
  • Built-in tools for creating a customized remastered ISO
  • Low resource usage and fast startup

Essora applications included

EssoraPup64 includes a collection of applications created to provide a consistent and integrated system experience. These tools are designed to remain lightweight while covering the most common desktop and system administration tasks.

Essora Store

Essora Store replaces Synaptic as the main graphical application manager in EssoraPup64.

Instead of requiring a different graphical tool for each package format, Essora Store provides a single interface for discovering, installing and managing software from multiple sources.

  • DEB packages from the configured system repositories
  • Flatpak applications
  • AppImage applications
  • Application search and category navigation
  • Visual application information and icons
  • Installed application detection
  • Software update management

This unified approach makes software management easier for new users while still providing access to the traditional Debian package ecosystem.

Essora Store is intended to be the primary graphical software manager in EssoraPup64, centralizing DEB, Flatpak and AppImage applications in one place.

Essora Installer

Essora Installer provides a clear and guided method for installing EssoraPup64.

Unlike traditional Puppy Linux installation tools, Essora Installer does not use Grub4Config. It uses the traditional GNU GRUB workflow commonly found in Linux distributions.

  • Traditional GNU GRUB installation
  • GRUB-PC support for BIOS systems
  • GRUB-EFI support for UEFI systems
  • Frugal installation support
  • Automatic copying of the required Puppy system files
  • Automatic generation of an EssoraPup64 boot entry
  • Integration with an existing Linux GRUB installation

The installer copies the required system files, prepares the installation directory and creates a boot entry compatible with a conventional Linux GRUB configuration.

Essora Network

Essora Network is the custom network connection manager included with EssoraPup64.

It provides a lightweight graphical interface for Ethernet and wireless connections and can remain available through the system tray.

  • Ethernet connection detection
  • Wi-Fi network scanning
  • Connect and disconnect actions
  • Wireless password entry
  • Saved network credentials
  • Connection status monitoring
  • System tray integration

Essora Bluetooth

Essora Bluetooth is a lightweight graphical utility for managing Bluetooth devices.

It can be used to search for nearby devices and manage connections without requiring a large desktop environment or a heavy Bluetooth frontend.

  • Bluetooth adapter detection
  • Nearby device discovery
  • Device pairing
  • Connect and disconnect actions
  • Support for phones, speakers and headphones
  • Simple GTK interface

Essora CPU Manager

Essora CPU Manager provides a graphical interface for viewing and controlling processor frequency behavior.

  • CPU information
  • Current frequency monitoring
  • CPU governor selection
  • Per-core configuration
  • Global configuration application
  • Persistent settings between sessions
  • Temperature and system information where supported

The utility is especially useful on laptops, older computers and systems where the user needs to choose between performance, power saving and lower operating temperatures.

Essora Firewall

Essora Firewall is a graphical firewall designed for Puppy Linux and EssoraPup64.

  • Simple firewall activation and deactivation
  • Network protection rules
  • Graphical rule management
  • Connection control
  • Lightweight system integration
  • No dependency on a large desktop firewall frontend

EssoraFM

EssoraFM is the custom file manager included with the Essora desktop.

It is designed to provide fast file navigation and common desktop file operations through a lightweight GTK interface.

  • Icon and list views
  • Directory navigation
  • File and folder management
  • Storage device access
  • Wallpaper selection
  • Desktop and system integration
  • Low resource usage

Essora PyMenu

Essora PyMenu is the main application launcher used by EssoraPup64 and EssoraWM.

The normal application menu is developed in C and GTK3 to provide fast startup and reduced memory usage.

  • Application categories
  • Fast application search
  • Favorites
  • User profiles
  • Theme support
  • Multiple language support
  • DEB application support
  • Flatpak application support
  • AppImage application support
  • Wine application support

PyMenu integrates with EssoraWM while remaining a separate and modular application.

Oply multimedia suite

EssoraPup64 includes several multimedia tools from the Oply application family. The applications are designed around simple interfaces and commonly used multimedia functions.

Oply

The main application in the Oply suite, providing access to its integrated multimedia tools and functions.

Oply Radio

A lightweight Internet radio player for browsing and listening to online radio stations.

Oply Video Player

A simple video player designed for everyday media playback without the complexity of a full multimedia center.

Oply Convert

A graphical multimedia conversion utility for converting common audio and video formats.

  • Simple multimedia playback
  • Internet radio support
  • Audio and video conversion
  • Lightweight graphical interfaces
  • Integration with the Essora desktop

Essora Cleaner

Essora Cleaner is a system cleaning utility designed to find and remove unnecessary files.

  • Application cache cleaning
  • Browser cache cleaning
  • Thumbnail cache cleaning
  • Temporary file removal
  • Flatpak residue detection
  • Support for the root account and regular users

The utility checks the locations used by Puppy Linux, Essora applications, browsers and Flatpak installations.

Essora Locale

Essora Locale provides a graphical interface for changing the system language, keyboard layout and regional configuration.

  • System language selection
  • Keyboard layout selection
  • Regional configuration
  • Puppy Linux locale integration
  • Multiple language support
  • Optional X session restart after applying changes

Essora VPN

Essora VPN provides a graphical interface for configuring and managing VPN connections.

Its purpose is to make VPN configuration more accessible without requiring the user to manage every connection manually from the command line.

  • Graphical VPN configuration
  • Connection and disconnection controls
  • Saved VPN profiles
  • Lightweight system integration

RemasterPup Essora

RemasterPup Essora is the custom remastering utility included with EssoraPup64.

It can create a new bootable ISO image from the currently configured system, allowing users and developers to preserve installed applications, system changes and personal customizations.

  • Create a new EssoraPup64 ISO
  • Preserve installed applications
  • Preserve system configuration
  • Include desktop customizations
  • Rebuild the main Puppy SFS files
  • Create personal or community editions

This makes EssoraPup64 suitable not only as a desktop operating system, but also as a base for creating specialized Puppy Linux distributions.

An integrated experience

One of the main objectives of EssoraPup64 is to reduce dependence on unrelated external utilities and provide applications designed specifically for the system.

Essora applications share a consistent visual appearance and are designed to work together. Many of the tools are developed in C and GTK3 to reduce memory usage, improve startup speed and minimize dependence on large runtime environments.

The system does not attempt to abandon the Puppy Linux philosophy. Instead, it preserves Puppy's speed, portability, flexibility and frugal installation model while adding a more complete and modern desktop environment.

Lightweight like Puppy, modern like Essora

EssoraPup64 combines Puppy Linux flexibility with XLibre, EssoraWM and a complete collection of integrated applications for software management, installation, networking, Bluetooth, multimedia, localization, system maintenance and ISO remastering.
