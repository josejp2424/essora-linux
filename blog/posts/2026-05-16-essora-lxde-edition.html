Essora LXDE Edition

Lightweight, full control and a new integrated panel

LXDE Edition Essora Panel 0.12-essora Kernel 7.0.0-rc7 Devuan · No systemd

Essora LXDE Edition represents an evolution of the classic desktop, combining the lightweight nature of LXDE with deep improvements in integration, performance, and system control.

This release introduces a key change: a new Essora Panel based on LXPanel, along with native tools specifically designed for the Essora ecosystem.

New Essora Panel (0.12-essora)

This edition includes a fork of LXPanel, fully adapted for Essora.

Main changes

  • Classic LXPanel menu removed
  • Direct integration with:
/usr/local/essora-kit/Pymenu.py
  • New identity: Essora Panel
  • Version: 0.12-essora
  • Full integration with the Essora system

Advantages

  • Faster and more dynamic menu
  • Full control over categories and icons
  • Independence from the traditional XDG menu system
  • Foundation for future enhancements

This change allows Essora to evolve without legacy limitations.

Updated Kernel

This version includes:

Linux Kernel 7.0.0-rc7

Providing:

  • Better support for modern hardware
  • Performance improvements
  • Enhancements in scheduling
  • Updated drivers and power management

Essora Native Tools

Essora includes a set of applications specifically developed for the system:

Essora Store

Package manager with support for:

  • DEB
  • Flatpak
  • AppImage

Optimized for performance and simplicity.

Essora VPN

Integrated tool for quick connection to private networks.

Essora Firewall

Simple and functional network control.

Essora Install

Custom installer that replaces Calamares, offering:

  • Better system integration
  • Greater control during installation
  • A more consistent user experience

Essora Remaster

Allows direct creation of a customized system ISO.

Essora CPUFreq

Advanced CPU management tool designed for Essora.

Built with a modern GTK3 interface, it provides:

  • Per-CPU frequency control
  • Governor selection: performance, powersave, balanced
  • Real-time monitoring
  • CPU temperature display
  • Apply settings globally or per core

It also includes multi-language support and a clean, user-focused interface.

Multimedia Suite: Oply

Oply is Essora’s integrated multimedia solution.

It includes:

  • Music player
  • Video player with MPV integration
  • Audio and video download from YouTube
  • Oply Radio with worldwide stations
  • Built-in IPTV support

All in a single application, maintaining a simple and functional interface.

Included Applications

The system includes a set of essential applications:

  • Firefox as the web browser
  • AbiWord for documents
  • Gnumeric for spreadsheets
  • Evince as PDF reader
  • GIMP for image editing
  • Viewnior as image viewer
  • PCManFM as file manager
  • Geany as code editor

Essora Philosophy

This release reflects the core philosophy of the project:

  • Lightweight and fast system
  • Full control over the environment
  • Reduced unnecessary dependencies
  • A consistent and self-contained ecosystem
  • Deep integration between components

Conclusion

Essora LXDE Edition is not just a lightweight desktop.

It is a reimagining of the classic environment, where every component is designed to work together, providing a cleaner, faster, and more controlled experience.

Essora continues moving forward with its vision:

“The new dawn of essentials.”
